Code running Directions :
1) Save the codes of Simple_ftp_server.py - receiver, Simple_ftp_client.py - sender and test.txt - sender in the working directory.

2)from Simple_ftp_server run the Simple_ftp_server.py with required args{ port, outputFilename, probability_loss}

3)from Simple_ftp_client run the Simple_ftp_client.py with required args{ host , port , file_name, N, MSS}

4)Once server is done receiving the file, it will continue to run and the code needs to be interrupted to terminate the execution.

5)Once the client is done the client will display the time spent in transmitting the file.
